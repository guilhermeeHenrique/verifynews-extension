# User Flow

1. User opens an online political article or page
2. The extension detects that the page contains political content
3. A visual reliability score appears
4. The user sees a quick status:
   - reliable
   - to verify
   - doubtful
5. The user clicks the extension for more details
6. A detailed panel or page opens
7. The user sees:
   - explanation
   - warning signals
   - sources
   - confidence level
